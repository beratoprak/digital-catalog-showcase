# Digital Catalog App (Showcase)

Satış temsilcilerinin Android tablet üzerinden firmalara ürün tanıtımı yapabilmesi için geliştirilmiş bir dijital katalog uygulamasıdır.

Bu repo **vitrin (showcase)** amaçlıdır.  
Gerçek proje, ticari ve güvenlik nedenleriyle private tutulmaktadır.

## Ne İşe Yarar?
- Ürünleri kategori bazlı listeleme
- Hızlı ürün arama
- Ürün detay ekranları
- Tablet üzerinden pratik sunum

## Kullanım Senaryosu
Satış temsilcileri, firmalara gittiklerinde tablet üzerinden ürünleri hızlı ve düzenli şekilde tanıtabilir.
